This project is a fully interactive **Power BI HR Analytics Dashboard** designed to help HR teams understand workforce behaviour, attrition trends, and performance insights. It answers key questions like:

* **Who is leaving and why?**
* **What factors influence performance?**
* **How do satisfaction and compensation shape retention?**
